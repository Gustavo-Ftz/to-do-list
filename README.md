# To-do List
Gerenciador de Tarefas simples, criado usando React e Taiwind CSS

Desenvolvido em **React e Tailwind CSS**, que permite adicionar, marcar como concluídas e remover tarefas.

## Funcionalidades  

- Adicionar novas tarefas  
- Marcar tarefas como concluídas  
- Remover tarefas individualmente  
- Interface simples, responsiva e de fácil uso  

## Tecnologias usadas  

- React
- Tailwind CSS
